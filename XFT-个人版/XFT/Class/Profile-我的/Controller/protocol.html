“融创西南幸福通”平台经纪人使用许可协议

一、总则

1.1 本协议是“融创西南幸福通”平台注册用户（以下简称“经纪人”）授权身份信息、使用“融创西南幸福通”平台小程序，以及享受“融创西南幸福通”平台小程序（以下简称“幸福通平台”）所提供服务的依据。

1.2 请务必仔细阅读和理解本协议中所规定的所有权利和限制。在经纪人注册前，需要仔细阅读并接受本协议的条款。经纪人完成注册，则视为经纪人已达成本协议的全部约定内容、成为融创西南幸福通经纪人。

1.3 幸福通将根据法律法规及其他实际情况之变动，在必要时对本协议的部分条款及规则予以完善与变更，经纪人需及时关注和了解各条款与规则的变动情况。如“幸福通平台”对部分条款及规则变更后仍然继续使用本平台，则表示经纪人同意并接受该等条款及规则的变更。融创西南区域集团及其关联公司（以下简称“融创西南区域集团”）将保留对本协议、操作规则及其他特别条款的最终解释权。

二、注册及认证

2.1 注册

经纪人需提供真实、正确、最新及完整的个人信息及资料，包括但不限于经纪人本人的姓名、联系电话、身份证、银行卡账号、通信地址、电子邮件等“幸福通平台”所需要的信息，一个姓名及身份证号只能申请注册一个幸福通平台账号，如遇换号情况，请拨打“400-9971918”申请解绑、更换。经纪人需重视所提供资料的重要性，因该等信息及资料的真实性、完整性瑕疵所导致的一切责任均由经纪人承担。

2.2 认证

经纪人同意根据“幸福通平台”的要求配合其对经纪人所提供的本人真实姓名、联系电话、身份证、银行卡账号信息等其他必要信息予以实名认证，未进行实名认证的，将影响经纪人相应佣金的结算与发放。同时因该等信息未经过实名认证所导致的一切责任均由经纪人承担。

三、交易规则

3.1 楼盘项目信息的准确性

经纪人在浏览“幸福通平台”中推出的各在售楼盘信息时，应当仔细阅读关于楼盘项目的名称、位置、户型、价格、佣金等，经纪人在向他人推介在售楼盘项目时应全面了解并准确传递所推介楼盘项目的各项信息及优惠活动，因经纪人自身原因造成的在推介过程中存在的信息偏差，应由经纪人承担全部责任，“幸福通平台”及融创集团对经纪人向任何第三方的不正当陈述不承担任何责任。

3.2 推荐客户

经纪人注册成功后，即可通过“幸福通平台”推荐意向客户购买具体的在售楼盘。经纪人需按照“幸福通平台”的要求，登记被推荐意向客户的相关资料，包括但不限于意向客户的真实姓名、联系电话、意向楼盘项目、预约看房时间、指定置业顾问等。

特别提醒：

1) 经纪人所推荐客户的真实姓名、联系电话、意向楼盘项目、预约看房时间信息极为重要，经纪人应确认所登记的客户相关资料的准确性。“幸福通平台”将以经纪人登记的意向客户联系电话作为首要标准进行系统的识别与判断，界定该客户推荐是否成功、认购是否成功、签约是否成功，及作为佣金结算的依据。

2) 如被推荐的意向客户对经纪人向“幸福通平台”提供其个人相关信息持有异议，或被推荐的意向客户与经纪人就信息提供产生任何纠纷的，均与“幸福通平台”及融创西南区域集团无关，所产生的责任均由经纪人承担。

3) 如被推荐的意向客户为融创集团员工、推荐者直系亲属或其本人，并产生成交，“幸福通平台”将不予结算佣金，并有权剥夺该经纪人继续享受“幸福通平台”所提供服务的权利。

3.3 被推荐客户的识别

被推荐客户信息登记完毕后，“幸福通平台”系统将根据所登记的信息自动识别判断，如符合本协议相关条款限定条件下（详见本协议中关于客户报备保护期、客户认购保护期等条款的限定）的登记客户，“幸福通平台”自动提示推荐成功；如该客户此前已成功被其他经纪人推荐且在保护期内的，系统自动提示推荐不成功。

特别提醒：

被推荐客户在经过身份验证后，有权向“幸福通平台”核实其信息在平台系统中的登记状态，不视为违反对任何经纪人的保密约定。如该客户明确表示拒绝被推荐，并要求“幸福通”系统取消其被推荐成功的状态，“幸福通平台”有权应客户的要求，取消其被推荐成功的状态。对上述已经过身份信息核验意向客户的知情权与决定权，“幸福通平台”及经纪人需予以尊重。

3.4 报备保护期

3.4.1 被推荐成功的意向客户，自“幸福通平台”系统提示推荐成功之日起若干日内（该期限为“报备保护期”，根据具体在售楼盘的实际情况，该期限可能存在差异，经纪人需密切关注所推荐客户报备保护期的失效时间），其他经纪人无法对同一楼盘的同一意向客户进行重复推荐。报备保护期届满，被推荐的意向客户仍未到访意向楼盘处的，其他经纪人可以对该意向客户进行重复推荐。

3.4.2 被推荐成功的意向客户，需在报备保护期内到访意向楼盘处，并配合售楼处相关销售人员做好该意向客户的信息登记工作，包括但不限于该意向客户的真实姓名、联系电话、意向楼盘项目、实际到访时间等。特别提醒，被推荐的意向客户实际到访时间不得早于自“幸福通平台”系统提示推荐成功时间，即经纪人须在客户到访前完成系统推荐，否则本次到访视为无效。

3.4.3 销售人员在系统内输入该意向客户的相关信息后，系统将进行自动识别判断，如与经纪人推荐的意向客户信息匹配且在报备保护期内，则系统显示到访成功，经纪人可通过“幸福通平台”查阅该意向客户的到访状态，如与经纪人推荐的客户信息匹配但报备保护期已届满，或意向客户提供的信息未能与任何经纪人推荐的意向客户信息相匹配的，系统将不显示到访成功的状态。

3.5 认购保护期

3.5.1 到访成功的被推荐的意向客户，自“幸福通平台”小程序提示到访成功之日起30个自然天内（经纪人需密切关注所推荐客户到访的意向楼盘项目中关于该期限的提示），其他经纪人无法对同一意向客户进行重复推荐。认购保护期届满，被推荐的意向客户仍未成功认购房屋并签署《认购书》的，其他经纪人可以对该意向客户进行再次推荐。

3.5.2 到访成功的被推荐意向客户，需在认购保护期内完成意向楼盘项目的房屋认购并签署《认购书》，并配合售楼处相关销售人员做好该客户的信息登记工作，包括但不限于该客户的真实姓名、联系电话、意向楼盘项目、实际认购签约时间等。

3.5.3 销售人员在系统内输入该认购客户的相关信息后，系统将进行自动识别判断，如与经纪人推荐客户信息匹配且在认购保护期内，则系统显示认购成功，经纪人可通过“幸福通平台”查阅该客户的认购状态；如与经纪人推荐客户信息匹配但认购保护期已届满，或认购客户提供的信息未能与任何经纪人推荐客户信息相匹配的，系统将不显示认购成功的状态。

3.6 成功签约

被推荐成功的客户，在到访保护期内现场到访意向楼盘项目，并在认购保护期内签署《认购书》，按照《认购书》约定时间及条件，签署《商品房买卖合同》并如期交齐首期款，并在房地产相关政府部门完成网签备案登记，申请公积金或商业按揭贷款的客户需提交已签署完成的《按揭贷款合同》及“幸福通平台”所需的其他相关材料，上述工作均完成后方可视为成功签约。

3.7 推荐佣金政策

3.7.1 经纪人推荐客户签约后，按照该客户实际购买总价的项目约定计佣比例获得佣金。经纪人应在领取佣金前完成“幸福通平台”实名认证，利于后期取得佣金。

1) 经纪人推荐客户时，需及时了解所推荐楼盘的佣金计发规则。

2) 以上计佣比例指“佣金点数”，该计算的佣金结果包含个人所得税所产生的费用。

3) 根据具体在售项目的实际情况，佣金结算点数可能存在时间差异，实际佣金点数以该房源认购时点载明的佣金点数为准结算。

3.7.2 原则上经纪人推荐的客户成功签约后，“幸福通平台”自动将经纪人实得之佣金金额划转至经纪人经过实名认证登记并绑定的个人银行账户。特别提醒，根据具体在售项目的实际，“幸福通平台”享有调整佣金具体支付方式的权利，调整后的佣金可能按期按比例予以支付。

特别说明：

1) 如不同的经纪人推荐的客户联名购买并成功签约的，以优先推荐的时间节点为准。

3) 如被推荐的客户拥有多个联系电话，“幸福通平台”仅以成功签约时客户登记的电话号码确定具有推荐关系的经纪人计发佣金，其他推荐该客户的经纪人不能获取佣金。

4) 如被推荐的客户在成功签约前因其他原因而获得额外低于正常销售价格的重大优惠幅度，且根据该额度外重大优惠幅度确定的房屋总价款签署《商品房买卖合同》的，推荐该客户的经纪人不能获取佣金。

5) 如有对客户归属判定具有争议，请联系该项目工作人员进行裁定。融创西南区域集团对客户归属争议及佣金判定归属具有最终解释权。

3.8 二级经纪人及其佣金政策

经纪人使用带有个人信息识别功能的“幸福通平台”小程序邀约注册二维码，成功邀约新经纪人注册并成为经纪人，则该经纪人为“一级经纪人”，其邀约的新经纪人为“二级经纪人”。当“二级经纪人”推荐客户成交，正常领取该项目佣金后，“一级经纪人”可额外获得“二级经纪人”佣金的20%奖励。员工不能被发展，成为他人“二级经纪人”，若已成为他人“二级经纪人”的，则不结算其上级佣金、只结算员工佣金。

3.9 分享奖励政策

经纪人或普通游客使用带有个人信息识别功能的楼盘分享图片，或转发楼盘至新用户，新用户首次登录“幸福通平台”小程序，则成功建立关联关系。若该新用户在首次授权48小时后，未被他方推荐成交、作为普通客户成交，则对分享此新用户的经纪人或普通游客发放该成交楼盘政策佣金的20%作为“分享奖励”。
